# 砲丸投げ

## ゲームの内容
スペースキーのみで遊べるハンマー投げを題材にしたゲームです。初めに、スペースキーを押してスタートします。
次に、下のタイマーがなくなるまでスペースキーを連打してパワーを溜めます。
パワーがたまり、タイマーがなくなったら、タイミングよくスペースキーを押して球を発射します。
ゲーム終了後もスペースキーを押すことでリセットすることができます。
上手い角度で発射できなかった場合や、うまくチャージできなかった場合はEscapeキーを押すことでリセットできます。

（プレイ中のスクリーンショットをドラッグ＆ドロップして追加する）

## 操作説明
- （例）矢印キー、若しくはAとDキーで左右に砲台を移動する
- （例）スペースキーを押すとビームを敵に向けて発射する
